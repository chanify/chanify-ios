# Chanify

[![iTunes App Store](https://img.shields.io/itunes/v/1531546573)](https://itunes.apple.com/app/id1531546573)

Chanify iOS Client.

## Build

```bash
$ pod install
```

## Test push in simulator

```bash
$ make apns text=12345
```

## Send message

```url
https://api.chanify.net/v1/sender/<token>/<message>
```
